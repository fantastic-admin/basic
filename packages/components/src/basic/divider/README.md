# FaDivider 分割线

带文字内容的分割线组件，支持文字位置调整。

## 使用场景

- 章节分隔
- 内容区域分割
- 表单分组标题
- 列表分隔
- 页面内容分区

## Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `position` | `'start' \| 'end'` | - | 文字位置（起始或末尾） |
| `class` | `HTMLAttributes['class']` | - | 自定义 CSS 类 |

## Slots

| 名称 | 说明 |
|------|------|
| `default` | 分割线中间的文字内容 |

## 注意事项

1. **文字居中**：默认情况下文字居中显示
2. **自动布局**：文字两侧的线条会自动填充剩余空间
3. **无文字时**：不提供 slot 内容时，显示为一条完整的分割线
4. **间距**：默认上下边距为 `1rem`（`my-4`）
