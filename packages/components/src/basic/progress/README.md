# FaProgress 进度条

线性进度条组件，展示任务的完成进度。

## 使用场景

- 文件上传/下载进度
- 任务处理进度
- 加载进度
- 安装/更新进度
- 表单提交进度
- 多步骤流程进度

## Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `class` | `HTMLAttributes['class']` | - | 自定义 CSS 类 |

### Model

| 名称 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `modelValue` | `number` | `0` | 进度值（0-100） |

## 注意事项

1. **进度范围**：进度值范围为 0-100
2. **动画效果**：进度变化时有平滑的过渡动画
3. **自定义样式**：可通过 CSS 类自定义高度、颜色等
4. **双向绑定**：支持 `v-model` 双向绑定
